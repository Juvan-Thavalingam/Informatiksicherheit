In diesem Repository dokumentiere ich praxisorientierte Labs aus dem Bereich IT-Sicherheit, in denen ich zentrale Sicherheitskonzepte technisch umgesetzt, analysiert und angegriffen habe.

Dabei habe ich kryptographische Sicherheitsmodelle bewertet, eine eigene PKI inklusive TLS-Zertifikaten aufgebaut, Netzwerkverkehr analysiert und Man-in-the-Middle-Angriffe durchgeführt. Zusätzlich habe ich eine Linux-Firewall mit nftables konfiguriert, Passwort-Hashes offline geknackt und Zugriffskontrollmodelle unter Linux praktisch umgesetzt.

Die Labs verbinden Angreifer- und Verteidigerperspektive und zeigen meine praktische Erfahrung in Netzwerksicherheit, Kryptographie, System-Hardening und Security-Analyse.
